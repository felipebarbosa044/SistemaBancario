<img align="center" width="40px" src="https://hermes.digitalinnovation.one/assets/diome/logo-minimized.png"> <h1>Sistema Bancário em Python</h1>

Projeto desenvolvido durante o **Bootcamp da DIO (Digital Innovation One)** com o objetivo de praticar conceitos fundamentais da linguagem Python.

## ✨ Funcionalidades

- Depósito
- Saque
- Consulta de extrato
- Controle de limite de saques diários
- Tratamento de exceções para operações inválidas
- Interface no terminal utilizando a biblioteca **Rich**

## 🛠️ Tecnologias utilizadas

- Python 3
- Rich (biblioteca)

## 📦 Instalação

Antes de executar o projeto, instale a biblioteca **Rich**:

```bash
pip3 install rich
```

## ▶️ Executando o projeto

Após instalar as dependências, execute o arquivo principal:

```bash
python __main__.py
```

## ⚙️ Configuração para o PyCharm

Para que as cores exibidas pela biblioteca **Rich** apareçam corretamente no terminal do PyCharm, siga o caminho:

**☰ Menu → Run → Edit Configurations... → Edit Configuration Templates... → Python → Modify options → Emulate terminal in output console**

Em seguida, marque a opção **Emulate terminal in output console**.
